These notes introduce [brms](https://paulbuerkner.com/brms/), a flexible R library to fit Bayesian models. Here, we show how to fit basic regressions,
read the summary outputs, and visualize results.

To benefit from these notes it is necessary to know how functions, vectors,
and data frames work in R, and to be familiar with linear regression. It is
convenient but *not* necessary to be familiar with Bayesian statistics.
